## Paths rule (important)

- Use only relative paths in HTML/CSS/JS.
- Do **not** use absolute paths like `/assets/...`.
- Correct examples:
  - `assets/css/styles.css`
  - `../assets/js/script.js`
  - `../../assets/img/logo.png`
